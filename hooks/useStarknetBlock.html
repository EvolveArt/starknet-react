useStarknetBlock

Hook for fetching a block.

Deprecation Notice

Use useBlock.

This hook fetches the latest block using the default provider. Block data is continuously refreshed in the background.

Examples

This example shows how to fetch the latest block.

function Component() {
  const { data, loading, error } = useStarknetBlock()


  if (loading) return <span>Loading...</span>
  if (error) return <span>Error...</span>
  return <span>Hash: {data.block_hash}</span>
}
