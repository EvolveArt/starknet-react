Hooks reference
Hooks reference and examples
useAccount

Hook for accessing the account and its connection status.

useAddressFromStarkName

Hook to get the address associated to a stark name.

useBalance

Hook for fetching balance information for ERC-20 tokens.

useBlock

Hook for fetching a block.

useBlockNumber

Hook for fetching the current block number.

useConnectors

Hook to work with connectors.

useContract

Hook to bind a Contract instance.

useContractFactory

Hook to create a ContractFactory.

useContractRead

Hook to perform a read-only contract call.

useContractWrite

Hook to perform a Starknet multicall.

useDeploy

Hook to deploy a Starknet contract.

useNetwork

Hook for accessing the current connected chain.

useSignTypedData

Hook to sign typed data.

useStarkName

Hook for fetching Stark name for address.

useStarknet

Returns the current Starknet context state.

useStarknetBlock

Hook for fetching a block.

useStarknetInvoke

Hook to invoke a smart contract method.

useTransaction

Hook to fetch a single transaction.

useTransactionManager

Hook to manage transaction across different components.

useTransactions

Hook to fetch a list of transactions in parallel.

useWaitForTransaction

Hook to fetch a single transaction receipt.
